Highlighted Research
Blur2Blur: Blur Conversion for Unsupervised Image Deblurring on Unknown Domains
Bang-Dang Pham, Phong Tran, Anh Tran, Cuong Pham, Rang Nguyen, Minh Hoai
CVPR, 2024
project page / arXiv

The paper introduces a novel framework to train camera-specific image deblurring algorithms by transforming challenging real blurry images into known blur-kernel only using unpaired data, simplifying the deblurring process and demonstrating superior performance in benchmarks.
HyperCUT: Video Sequence from a Single Blurry Image using Unsupervised Ordering
Bang-Dang Pham, Phong Tran, Anh Tran, Cuong Pham, Rang Nguyen, Minh Hoai
CVPR, 2023
project page / paper / video / poster

Our work tackles image-to-video deblurring, resolving sequence order ambiguity using a self-supervised method. We also introduce a diverse real blur2vid image dataset (RB2V) for this task, covering domains like faces, hands, and streets.
